# Security Policy

## Supported versions

This project is deployed continuously from the `main` branch to
[zelda.ifq.ai](https://zelda.ifq.ai). Only the latest `main` is supported.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Instead, report privately via GitHub Security Advisories:

1. Go to the repository's **Security** tab.
2. Click **Report a vulnerability** (Private vulnerability reporting).

Alternatively, reach the maintainer through [github.com/peixl](https://github.com/peixl).

We aim to acknowledge reports within 72 hours and to provide a remediation
timeline after triage. Thank you for helping keep the project and its users safe.
